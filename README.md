## 🧩 各数据结构核心原理    
### 1. 单链表    
- **核心**：用单个节点存储数据+next指针，通过指针串联节点    
- **关键操作**：头插/尾插、删除节点、反转链表    
- **注意点**：删除节点时必须保存前驱指针，防止内存泄漏    
    
### 2. 栈    
- **核心**：先进后出（LIFO）    
- **数组版**：用可扩容数组+栈顶指针实现，时间复杂度O(1)    
- **链表版**：用单链表头插实现，无需扩容，时间复杂度O(1)    

### 3. 队列    
- **核心**：先进先出（FIFO）    
- **循环队列（数组版）**：    
  - 判空：`head == tail`    
  - 判满：`(tail + 1) % size == head`（牺牲一个空间）    
  - 元素个数：`(tail - head + size) % size`    
  - 入队：`tail = (tail + 1) % size`    
  - 出队：`head = (head + 1) % size`    
- **链式队列（链表版）**：    
  - 核心：头删尾插，用front/rear双指针保证O(1)操作    
  - 无需扩容，适合动态长度场景
